---

## ✨ Возможности

- 🔔 **Всплывающие уведомления** — появляются поверх всех окон
- 🎨 **6 цветов оформления** — настрой под себя
- 📝 **Описание задач** — до 200 символов
- 📋 **История** — завершённые задачи хранятся 24 часа
- 🚀 **Автозагрузка** — запускается вместе с Windows
- ⚡ **Лёгкий** — меньше 50 МБ памяти
- 🔄 **Автообновления** — всегда свежая версия

---

## 📥 Установка

1. Скачай `QuickReminder.exe` из [Releases](https://github.com/ekadvladsheglov-tech/QuickReminder/releases/latest)
2. Запусти (не требует установки!)
3. Заполни регистрацию
4. Готово! Иконка появится в трее 🔔

## 🎮 Управление

| Действие | Как сделать |
|----------|-------------|
| Создать напоминание | Вкладка «Создать» → заголовок → время → Добавить |
| Показать окно | Клик на иконку в трее |
| Настройки | Кнопка ⚙️ |
| Тест уведомления | Кнопка «Тест уведомления» |
| Обновление | Кнопка 🔄 |

## ⚠️ Антивирус может ругаться

Это **ложное срабатывание**. Программа безопасна.

**Причина:** EXE собран через PyInstaller. Некоторые антивирусы 
помечают такие файлы как подозрительные.

**Решение:** Добавьте QuickReminder.exe в исключения антивируса.

**Исходный код открыт** — вы можете проверить его на GitHub.
